# VPSOptimizer

一个开源的一键 VPS 网络优化工具箱，支持自动开启 BBRv3、网络 sysctl 参数优化、测速工具安装，并输出优化报告。

## 功能
- 自动检测内核版本，尝试启用 BBRv3
- 应用推荐的网络优化参数
- 安装 iperf3 进行测速
- 更多功能即将添加...
